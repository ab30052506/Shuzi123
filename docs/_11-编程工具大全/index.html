        # 程式設計師必備軟體工具推薦

        3000+ 程式設計師效率工具合輯，點擊即用

        你好，我是程式設計師魚皮。

        在 程式設計導航 (opens new window) 群裡經常看到有同學問，畫流程圖、架構圖用什麼工具？剛學程式設計，建議下載哪個程式碼編輯器？AI 工具到底哪個好用？

        日常總卡在「找工具」這一步，實在心累。

        於是我們上線了【程式設計資源工具大全】，這裡一站式匯總了程式設計師全工作流所需的各類神器，即搜即用，大幅提高工作效率！

        # 一、開發工具

        匯總 600+ 常用程式設計工具和資源，涵蓋主流開發環境 IDEA、VS Code，開發框架資源文件 React、Vue，程式碼託管平台 GitHub、Gitee 和開源鏡像清華源、華為源，大幅提高開發效率。

        # 推薦工具 Top 5

        🏅 Top 1：Git

        分散式版本控制系統，程式碼版本管理核心工具，團隊協作開發基礎，因此作為程式設計師必備工具之首。

        🥈 Top 2：Visual Studio Code

        輕量級跨平台程式碼編輯器，強大的插件生態，支援多種程式語言，2025 年依然是程式設計師最熱編輯器。

        🥉 Top 3：Cursor / Claude Code

        以 AI 為核心的程式碼編輯器，代表未來程式設計趨勢。

        Top 4：JetBrains IDE 系列

        專業的整合開發環境，提供 IntelliJ IDEA、PyCharm、WebStorm 等全家桶軟體，具有智能程式碼分析能力。

        Top 5：Navicat / DataGrip

        資料庫管理工具，能夠可視化操作資料庫，支援 MySQL、PostgreSQL、MongoDB 等。

        # 二、AI 工具

        整理 1600+ AI 工具，包含 AI 編輯器 Cursor、Trae、AI 生圖工具即夢、可靈 AI 和大模型平台 ChatGPT、Gemini、DeepSeek，讓 AI 助力工作。

        # 推薦 AI 工具

        AI 程式設計工具：Cursor、GitHub Copilot、Codeium、Windsurf

        AI 對話工具：ChatGPT、Claude、通義千問、DeepSeek、Kimi

        AI 繪圖工具：Midjourney、Stable Diffusion、即夢、可靈 AI

        AI 寫作工具：Notion AI、秘塔寫作貓、火山寫作

        AI 影片工具：Runway、Pika、剪映

        # 三、設計工具

        提供從 0 到 1 的設計工具合輯，產品原型設計 Axure、Draw.io，平面創意設計 Canva、稿定設計和配色工具 Coolors、Color Hunt，覆蓋產品開發工作流。

        # 推薦設計工具

        原型設計：Axure、墨刀、即時設計

        繪圖工具：Draw.io、ProcessOn、億圖圖示

        平面設計：Canva、稿定設計、創客貼

        配色工具：Coolors、Color Hunt、中國色

        # 四、測試和專案管理工具

        精選測試和專案管理工具，包含自動化測試 Selenium、API 除錯 Postman、Apifox 和主流專案管理平台禪道、Jira，助力團隊高效協作和交付。

        # 推薦工具

        API 測試：Postman、Apifox、JMeter

        自動化測試：Selenium、Cypress、Playwright

        專案管理：禪道、Jira、Trello、飛書專案

        團隊協作：企業微信、飛書、釘釘

        # 五、運維部署

        整合主流雲平台與和運維部署工具，涵蓋容器和運維 Docker、Nginx、邊緣部署平台 Vercel、Cloudflare 和雲服務平台阿里雲、騰訊雲、華為雲。

        # 推薦工具

        容器技術：Docker Desktop、Kubernetes

        Web 伺服器：Nginx、Apache、Caddy

        部署平台：Vercel、Railway、Sealos、Cloudflare Pages

        雲服務：阿里雲、騰訊雲、華為雲、AWS

        終端工具：Xshell、FinalShell、iTerm2、Windows Terminal

        # 六、效率工具

        除了開發相關的工具，還有一些能提升工作效率的通用工具，基本上魚皮都用過。

        • Postman / Apifox / JMeter：API 介面除錯、自動化測試、文件生成；JMeter 提供專業壓力測試和效能分析

        • VMware Workstation / VirtualBox：虛擬機器軟體，多環境開發測試必備

        • Docker Desktop：容器技術，簡化應用打包、分發與執行

        • Xshell / FinalShell：SSH 終端工具，遠端伺服器連接管理

        • Mac iTerm2 / Windows Terminal：終端工具，更好的終端體驗

        • Homebrew（Mac）/ Chocolatey（Windows）：套件管理工具，快速安裝各類軟體

        • WinSCP / FileZilla：檔案傳輸工具，FTP/SFTP 客戶端

        • Fiddler / Charles / Wireshark：抓包除錯工具 + 網路協定分析

        • Notion / 語雀：文件知識管理平台，團隊協作文件編寫

        • Typora / PicGo / mdnice：Markdown 編輯器、圖床管理工具、公眾號排版神器

        • 企業微信 / 飛書：團隊溝通協作平台

        • uTools / Alfred：效率工具啟動器，快速啟動應用和工具

        • Snipaste：截圖貼圖工具，支援貼圖顯示和標註

        • Everything：檔案搜尋神器，秒速定位所需檔案

        • 剪貼助手：跨平台剪貼簿管理工具，多裝置同步（https://jianqiezhushou.com）
