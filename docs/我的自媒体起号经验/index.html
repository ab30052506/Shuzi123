              # 我的自媒體起號經驗

              從 0 到 10 萬粉的真實經歷

              大家好，我是魚皮。剛來 X 平台，關注數就突破了幾千，說實話有點受寵若驚，感謝大家助力每一個夢想 🌹。

              很多朋友問我是怎麼做到的？

              除了我之前有一些國內平台影響力外，更重要的還是方法，但這個方法並不難，人人都可以做到。而且我當初在各個平台起號時，基本都是這麼做的。

              看著突突漲的關注數，我想起了自己最初從 0 到 10 萬粉的那段時光。今天想跟大家聊聊這部分故事，以及分享一些實實在在的起號經驗。

              無論你是想通過自媒體推廣自己的產品，還是想建立個人品牌，這些經驗都能幫到你。

              # 從 18 播放量開始

              2020 年特殊期間，我在家裡遠程 TX 實習。有一天打開抖音，隨便翻了幾個視頻後，我發現在抖音，並沒有多少做技術分享的程序員。或許是因為抖音偏娛樂向？又或許是因為短視頻的限制，很難通過隻言片語講清一個技術？

              我是一個喜歡嘗試新事物的人，正好錄製短視頻的成本也並不會很高，於是我就在工作之餘開始了自己的短視頻製作生涯。

              由於我在大學期間獨立開發過一個面試刷題產品 —— 面試君主（沒錯，這是我現在幾十萬用戶產品 - 面試鴨的原型），因此第一個抖音短視頻賬號就叫 “面試君主”，視頻內容專注於程序員高頻面試題解析，正好幫助我自己春招獲得更多機會。

              真正做短視頻後，我才發現事情並沒有自己想像的那麼簡單。

              首先是製作成本，做短視頻並不只是簡簡單單用手機拍攝一下、錄個屏就完事了，除了選題和最基本的內容外，很多的素材都需要精雕細琢，比如封面、標題、標籤、第一眼、開頭等。我會先寫稿子、打磨和驗證內容，然後再錄像，結果第一期視頻就做了三四個小時。

              功夫不負有心人啊！猜猜最後的播放量是多少？

              我記得很清楚，一天下來，播放量是整整 18！

              我想過新人播放量會比較低，但沒想到 TNND 這麼低！

              說不難受是不可能的，但畢竟是第一個視頻；而且我當時的目的很純粹，就是通過分享，在幫助大家的同時鞏固自己的知識。

              因此，我沒有放棄，又一連堅持了大概一個月，每天至少更新一個視頻，週末還會更新 3 個視頻！

              雖然每個視頻的播放量也只有幾十，但是一個多月的堅持，也讓我收穫了人生中第一批粉絲，大概有 100 多個，其中幾十個還是我的朋友。最高播放量的一個視頻達到了 1800。

              所以我很喜歡 18 這個數字，它對我很有意義。

              # 第一次爆款，被噴慘了

              後來有一天，我靜下心來思考，這樣的堅持是否還具有意義？

              於是我去看了很多其他同學的視頻，發現往往 真實的故事最能打動人。

              於是，我又註冊了一個新的抖音號，也是現在的號【程序員魚皮】，並且第一次通過短視頻平台將我的編程學習和求職經歷分享給大家。

              結果，這次的播放量和點讚量出乎我的意料，第二天醒來，消息已經 99+，當時的心情還是非常激動的。

              但是，當我打開評論區，心情立刻就跌入低谷。評論區充斥著各種各樣的 懷疑、鄙視、甚至是謾罵，很多人說 “又一個營銷號誕生了”，還有什麼 “抖音各個都是人才，把外行人忽悠的團團轉”。

              好吧，我算是知道這個視頻為什麼能火了，雖然都是差評，但這 互動率 是真的高。

              記得當時有一個罵我的評論點讚數還巨高，都快接近我當時的視頻點讚數了。現在想想，真的要感謝當時罵我的朋友們，當然也包括現在的。

              這條視頻過後，我開始思考，如果要分享編程技術，也可以換一種更有趣、更迎合大眾、更容易被接受的方式。創作當然可以只顧著自己爽，但是如果能讓觀眾也爽了，豈不是一舉兩得？

              於是，我開始分享一些優質的、易用的編程資源，以及一些自己積累的編程小知識。並在這個期間不斷地學習短視頻創作技巧，學會看數據，學會優化各項指標，多看他人的視頻來學習好的創意和經驗。

              還記得大學上計算機課時老師講的一句話：1.01^365 次方等於多少？每天改變一點點，也會有巨大的變化，可怕的是盲目堅持，而不做任何改變。

              # 第一次直播，阿巴阿巴

              接下來，我嘗試了人生中第一次直播，畫面是一台電腦，記得當時大概有十幾位觀眾，我不知道該講啥，也不知道這十幾位觀眾在看啥，我就一直在重複 “大家好，這是我第一次直播”，結果還真的有人一直看。

              後來，我堅持每晚直播一小會兒，通過在直播間和大家交流編程，我感受到了多樣人生，和粉絲朋友們保持了良性互動，也幫助我收穫了更多的粉絲。

              而且你敢想像麼，我是個 i 人（ISTJ），看過我吐槽公司視頻的朋友應該能感受到。

              通過不懈的努力，我的抖音漸漸有了起色，粉絲數和播放量都穩定增長。雖然說抖音是一個偏娛樂和生活的平台，但是我在這裡也擁有了一片屬於自己的小天地，認識到了很多的朋友。

              堅持也讓我看到了另一個變化，就是評論區底下對我的懷疑和謾罵變得越來越少。當然即使是現在，依然會有很多人罵我，但這不是很正常的事情麼？我又不是 money 人見人愛~ 看到自己一直在堅持的事情幫助到了更多同學、得到了肯定，這就足夠了。

              # 多平台擴展

              有了抖音的基礎後，我開始嘗試其他平台，比如公眾號和 B 站，都叫程序員魚皮。

              做 B 站的時候，我正式發布的第一個漲粉視頻想必大家也能猜到了。沒錯，就是我自己從學編程到入職大廠的個人經歷。具體漲了多少粉我忘了，但印象中這一個視頻基本上就把號帶起來了。讓我記得最清楚的是，有小夥伴發現我是個歪嘴戰神，哈哈哈。

              但公眾號的漲粉沒有那麼容易，剛開始我傻傻地以為想要漲粉，就要多寫文章，於是給自己定下了每天 1 篇的目標。結果質量不高、閱讀不高、粉絲也沒漲幾個。

              很快，我意識到這樣下去得完犢子，必須要改變方法！

              於是我開始瘋狂混圈子，向成功的前輩交流、學習、模仿；找大號投稿，爭取轉載機會；開拓渠道，把同一篇文章、同一個視頻發布到多個平台；利用抖音給公眾號引流等等。關於我做公眾號從 0 到 10 萬粉的經歷，後面會給大家更詳細地分享。

              就這樣，總共大概過了 1 年左右，抖音粉絲 7.8 萬、公眾號 1.8 萬、再加上 B 站，差不多累積 10 萬粉絲 了，第一個小目標達成。

              所以看到這裡，你應該已經知道我開頭說的起號方法是什麼了吧？

              沒錯，就是 分享真實的經歷。

              從短視頻 18 播放量 -> 棄號重開 -> 第一次小火 -> 堅持直播 -> 擴展圖文和長視頻，我一直在做的，是把自己真實的經歷、真實的想法、真實的感受分享給大家。

              這次來到 X 平台，我看到剛做 X 的朋友 @程序員小灰 哥通過分享真實經歷快速漲了很多關注，這更堅定了我對這種方式的信心。

              我覺得這是每個人都能做到的事，不需要華麗的辭藻、不需要規整的文章結構，只需要把自己的故事真誠分享出來就好。

              你的經歷絕對是獨特的，至於剩下的，就交給算法吧。

              下面我想結合這段經歷，跟大家分享一些實實在在的自媒體起號建議。

              # 一些自媒體起號建議

              1）真實最重要

              縱觀自己和全網的觀察，我發現 越真實、越自然，越容易爆火。

              做好內容，是一切的基礎。真實的東西在 AI 時代會更可貴。不要刻意模仿別人，你的經歷就是最獨特的素材。

              2）降低預期，堅持行動

              我知道很多朋友會因為流量感到焦慮，覺得自己做的東西沒人看、沒人認可。

              但我想說：堅持下去一定會成功。

              我剛開始做公眾號時，閱讀量也是個位數。但之後我連續 100 多天下班後直播到凌晨，就是這種堅持，讓我突破了瓶頸。

              面對焦慮最好的辦法就是：降低預期並且堅持行動，我真的很喜歡 Speak Less Do More 這句話。

              3）有趣很重要

              對於高級編程技術或完整教程，真正要看這些內容的人會主動搜索，不需要你來把文字稿整成視頻。尤其是 AI 時代來了，有些追求效率的朋友甚至不會去看長視頻，而是直接用 AI 工具總結了。

              建議把內容做得輕鬆有趣，讓大家在零碎時間了解新知識。像我就盡量讓每期視頻都能給大家帶來一些知識，哪怕是 “我的伺服器被攻擊了” 這種難繃的經歷，相信也能讓大家少踩踩坑。

              4）AI 只是工具，核心是你的創意

              現在通過 AI + 各種工作流可以快速批量創作內容，雖然有機會起量，但一定有對應的瓶頸，因為越簡單的東西，複刻成本越低。

              關鍵還是要有自己的創意和特色，只不過如今實現創意的成本變得簡單了。

              5）立刻行動

              很多人問我：“我也想做自媒體，但不知道怎麼開始。”

              我的回答是：不要光想、多去嘗試！

              我自己就很討厭那種理論派，一套一套的，看了就忘。反而是自己練號的過程中，會自然地慢慢學到很多知識，而且印象更深刻。舉個例子，被限流一次你就知道下次文案中要注意哪些違禁詞了。

              還有怎麼寫標題、怎麼做封面、怎麼選題、怎麼看數據、怎麼優化內容，這些都是在實踐中學會的。

              練廢了大不了重來嘛！

              像我第一個賬號 “面試君主” 就是練廢的，但那段經歷讓我學到了太多東西，也是之後新號起量的關鍵。

              6）找準定位，做自己擅長的

              不要去跟某一領域更專業的團隊競爭。比如我剛開始不會去出幾十個小時的完整課程，因為培訓機構更專業。

              像我前期的定位很清楚：分享真實經歷 + 有趣的技術知識 + 優質資源推薦。

              找到自己的定位，做自己擅長的事，這很重要。

              說到這裡其實有個讓我很氣的事，我 21 年 ~ 22 年期間自己做了很多有趣的項目，結果到後面發現觀眾根本不在意項目是不是你開發的，有用 / 有趣就夠了，很多時候推薦自己費力開發的項目還不如推薦別人的播放量要高。

              7）時間管理

              還記得之前有個罵我的小伙子，說我背後肯定有團隊，1 個人怎麼可能做那麼多事？

              但當時真的沒有，我只是用了一些時間管理技巧罷了。

              首先我最理想的狀態是把工作中的發現、學到的技術進行分享，而不是從 0 開始分享自己完全不知道的東西。

              有人問我的時間哪來的？

              其實為了做自媒體，我放棄了午休、放棄了遊戲、放棄了很多出去玩耍的機會，也熬了不少夜。我估計 90% 的同學沒時間的原因都是在玩遊戲和回消息（水群）吧，如果不玩遊戲，時間肯定還是足夠的。

              而且一旦你像我一樣把自媒體當做興趣和愛好，而不是任務，漲粉會變得容易很多。

              8）接受焦慮，享受過程

              Q：做號過程中會缺乏動力麼？

              A：肯定會的，這時要給自己找點刺激。心態要隨緣，刻意會不持久。

              Q：你會有數據會焦慮麼？

              A：肯定會的，我會經常和別人對比，也曾一度懷疑自己被限流（雖然直到現在我依然懷疑哈哈）。但更多時候，我關注的還是內容本身，學習別人作品中優秀的地方，縮小差距。

              最後，如果你也想做自媒體，我的建議是：

              別想那麼多，先發第一條內容，告訴別人是誰。

              哪怕只有 18 播放，那也是一個開始，至少你種了一顆種子，不是麼？

              這次來到 X，如此快速的關注對我來說不一定是好事，因為在這裡我還是新手，還請各位大佬們多多擔待。

              如果你支持我，那麼我會很開心，謝謝你；如果你 DISS 我，把我當成一條臭狗，那我也希望能通過之後的分享打破質疑。我會盡力在這裡多分享自己特有的內容，希望能給大家一些啟發，讓我們一起成長！🚀
