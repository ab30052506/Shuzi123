    # 我的自媒體漲粉運營之路

    從 0 到 200 萬粉的真實經歷

    大家好，我是魚皮。98 年的小弟，之前在騰訊全幹開發，現在是全網 200 萬粉的 AI 編程知識博主 + 創業者。

    這篇文章，我想跟大家聊聊我這 9 年的經歷：從大學自學編程 → 女朋友跟別人跑了 → 進入騰訊 → 離職創業 → 做自媒體從 0 到 200 萬粉 → 產品破產 → 公司產品月活過百萬。有高光時刻，也有至暗時刻；有成功的經驗，也有血淚的教訓。

    如果你：

    • 想通過自媒體快速漲粉或靠副業提升收入
    • 想知道如何從 0 開始做起一個事業
    • 對編程學習、求職、AI 技術感興趣
    • 正在創業或想創業，需要一些真實的經驗參考

    那這篇文章應該能給你一些啟發。

    如果你之前完全沒聽說過、或者只是半路關注，那正好，讓我給你講一個從普通程序員到自媒體創業者的真實故事。

    # 我的故事

    # 大學：女友跑了，但我拿到了第一

    16 年進入大學後，我開始自學 Java、前端、Python、Go 等多種編程技術。

    那時候的我，真的是個卷王。白天上課，晚上泡在圖書館敲代碼，凌晨回宿舍是常態。作為工作室負責人，我帶隊給學校建設了幾十個網站，用賺的錢給自己買了第一台顯示器，那種靠自己 “手藝” 賺錢的感覺，真的很爽。

    但代價也是慘重的。

    長期高強度學習 + 熬夜，我的腸胃出了問題，有段時間飯都吃不下去，只能靠漢堡炸雞外賣度日。更慘的是，女朋友因為我天天不陪她、只顧著敲代碼，最後跟別人跑了。

    那段時間真的很痛苦，我甚至懷疑過自己這麼拼到底值不值得。

    但既然選擇了這條路，就沒有回頭的餘地。我繼續埋頭苦幹，最後以專業第一的成績拿到了國家獎學金、挑戰杯競賽國獎和上海市特等獎、上海市優秀畢業生。大三還跟老師出版了《區塊鏈智能合約技術與應用》教材。

    現在想想，那種為了一個目標全力以赴的感覺，真的很純粹。雖然失去了很多，但也收穫了更多。

    # 騰訊：211 本科生在清北堆裡的逆襲

    19 年校招時，我拿到了多家大廠 offer，最終選擇騰訊。

    說實話，當時壓力很大。我只是 211 本科，身邊的實習生清一色清北復交的大佬，學歷、背景都比我強。記得第一次公司內部封閉培訓時，當我介紹學歷時，大家看我的眼神就像看一條 “臭狗”。

    但我沒有失去信心，既然學歷拼不過，那我就拼時間、拼努力。別人下班了，我還在公司學習內部文檔；別人周末出去玩，我在研究項目代碼。最拼的那幾天，因為學得太晚沒有地鐵了，我甚至買了個行軍床，直接住在公司。

    住在公司的體驗真心差，保潔阿姨大早上 6 點多就會把你弄醒

    功夫不負有心人，我最終以組內第一的成績通過了轉正答辯，並斬獲 SSP 最高級別的 offer。

    在職騰訊近 4 年，我主導 BI 項目研發、參與大數據研發，曾獲騰訊內部應用開發競賽冠軍、5 星優秀員工、晉升綠色通道。工作兩年半就擔任了新人導師。

    剛工作後，我用賺的錢給家裡買了台電視、給爸媽買了按腳儀。那一刻，我覺得所有的付出都值了。

    # 自媒體：從抖音 18 播放量到全網 200 萬粉

    我最開始的自媒體嘗試是做短視頻。

    那是疫情期間，我在抖音上開了個賬號叫 “面試君主”，專門分享程序員面試題。第一個視頻我做了三四個小時，結果播放量只有整整 18！

    雖然打擊很大，但我沒有放棄，一連堅持了一個多月，每天至少更新一個視頻。最高播放量的視頻達到了 1800，收穫了人生中第一批 100 多個粉絲。

    後來我換了個思路，註冊了新號【程序員魚皮】，分享自己的真實經歷。結果第二天醒來，消息 99+，我超級激動！

    但打開評論區，心情立刻跌入谷底：清一色的懷疑、鄙視、甚至謾罵。“又一個營銷號”、“騙人的吧”...... 那些惡評點讚數都快接近我視頻點讚數了。

    當時我的心理素質不夠強，就刪掉了評論。現在想想，真的要感謝當時罵我的朋友們，因為互動率真的高啊（笑）。

    20 年國慶期間，我做了另一個決定。

    那天我剛打完遊戲，無所事事。進騰訊一年後，我發現自己似乎已經懶得做工作之外的事情了，每天下班後還耗費精力在遊戲世界裡渾渾噩噩。

    我想，有沒有一種外力，可以讓我重燃大學時奮鬥的熱情？

    想到這裡，我遊戲裝備都懶得賣了，立刻打開公眾號《程序員魚皮》，寫下了第一篇文章。

    從那天起，我再也沒時間賣遊戲裝備了。

    剛開始我傻傻地以為想要漲粉，就要多寫文章，於是給自己定下了每天 1 篇的目標。在粉絲沒幾個的情況下，硬是生懟了十幾篇文章，質量不高、閱讀不高、粉絲也沒漲幾個。

    很快，我意識到這樣下去得完犢子，必須要改變方法！

    於是我開始混圈子，向成功的前輩交流、學習、模仿；開拓渠道，把同一篇文章、同一個視頻發布到多個平台；甚至人生第一次直播，十幾個觀眾，我不知道該講啥，就一直重複 “大家好，這是我第一次直播”。

    最狠的是，我曾經連續 100 多天，每天下班後直播到凌晨帶大家敲代碼。

    說真的，壓力很大。聽說當程序員容易脫發、自媒體容易脫發，那這兩個加起來豈不是發如雨下？我的頭髮是真真實實地在持續哭泣！

    但我相信：看似一個人的火爆是偶然，對於知識創作者來說，更多的是 厚積薄發。有東西的人，就是有東西，只需要打開一個開關，光芒就會綻放。

    就這樣，靠著堅持和死磕，至今累計創作近千篇、千萬字，全網積累了 200 萬+ 的關注。

    後來評論區的懷疑和謾罵越來越少，甚至到現在很少再出現差評了。有時候我還挺想念那些罵我的日子，畢竟互動率真的高（開玩笑）。

    # 知識付費：從質疑到認可的 3 年

    22 年，我開通了知識付費圈子，也就是現在的 編程導航 (opens new window)。

    剛開始有很多質疑，我完全能理解。但我堅持了下來，因為我不想做那種收割一波粉絲、賺一桶金就不再維護內容的事，良心會痛。

    這 3 年來，我帶大家做了十幾套完整的原創項目教程，包括 AI 應用開發、微服務架構、高並發系統、輪子項目等等。很多項目都是十幾萬字的文字教程 + 幾十小時的視頻講解，期間還出版過項目實戰教程書籍，我是真的在用心做這件事。

    很多同學通過這些項目學習，成功拿到了大廠 offer。每次看到大家上岸的消息，我真的超開心。

    如今 編程導航 (opens new window) 已經成為國內最大的編程學習交流圈之一，累計幫助幾十萬名程序員朋友提升技術和項目能力。

    # 創業：從 5 萬用戶到破產，只用了一個月

    23 年，我做了人生中最重要的一個決定：離職創業。

    原因有 2 個：

    1）AI 風口來了，我想試一試

    2）在公司成長空間有限，我想做做自己感興趣的事。

    當時公司像個籠子一樣，框住了我這隻小鳥兒。我有很多想法，但工作佔用了我每天絕大多數時間，甚至還有同事舉報我做自媒體。

    我不想被束縛，我想做自己熱愛的事情。

    24 歲那年，我組了個 4 人小隊，在毫無經驗的情況下，開啟了一場艱難的創業之旅。

    老實說，創業這兩年多，我經歷了之前 20 多年人生都從未有過的打擊，甚至比高考的失利還要大。

    剛開始，我熱情高漲、信心滿滿。帶著僅有的一名前端同學從 0 開始做了個 AI 助手平台《魚聰明 AI》，用了一個月左右上線。

    結果很順利，上線一周用戶數就突破了 5 萬！我開始幻想自己能夠成為 “李” 爾蓋茨了，甚至還帶著同事們去北京團建遊玩。

    但變化來得太突然了。

    僅僅一個月後，產品就破產了，入不敷出！

    AI 產品遭到封殺、同類產品瘋狂內卷、前期白送了大量會員被薅羊毛...... 這給了我沉重的打擊，畢竟這是公司的第一個產品、做了這麼久的產品、給予了這麼大希望的產品，突然就破產了。

    都說要擁抱變化，我特麼的擁抱不住啊！！！

    後來我冷靜下來，意識到在沒有穩定團隊的情況下，追求爆款的代價太大了，不如踏踏實實做點有長期價值的事。

    於是我們開始做各種工具類產品：老魚簡歷 (opens new window)、代碼小抄 (opens new window)、剪切助手 (opens new window)、面試鴨 (opens new window) 等等。同時持續完善編程導航，帶大家做了十幾套完整的項目教程。

    這期間的壓力大家可能無法想像。

    每月巨額的開銷、團隊的磨合、產品的迭代、市場的競爭、惡意的詆毀、網暴、舉報、造謠、欺詐...... 這些電視劇裡才會出現的劇情真實地發生在了我身上。

    還有身體的崩潰。長期熬夜、直播耗氣、飲食不規律、精神壓力大，導致我的腸胃經常疼痛，甚至專門住院做了檢查。頭髮也掉得賊拉快，直接上圖：

    我記錄下當時的日記，用一個詞形容就是 “麻了”。各種事情帶來的精神壓力真的太大了，就像達摩克利斯之劍一直懸在頭頂。

    毫不誇張，那段時間我 emo 的天數可能比整個 “前半生” 都要多。

    但我沒有放棄。

    剛創業時，我很害怕踩坑、害怕失敗。但是經歷了越來越多的失敗後，我發現自己竟然有點兒 “上癮了”！因為每次失敗，都會有相應的成長：

    • 產品破產，讓我有了更多對產品和商業的思考
    • 招人不慎，會讓我更謹慎地調整招聘標準
    • 合同被違約，會幫我們提升法律意識

    現在的我，越來越享受踩坑的過程，心態也更穩定了。我經常安慰自己：哥就是來交學費的嘛，哪怕最後公司幹黃了，也是一段獨特的經歷不是麼？

    # 現在：好像又回到了原點

    如今，我自己的科技公司魚鳶網絡，團隊已經 10 多人。

    經過兩年半的沉澱，我們做出了編程導航、面試鴨、老魚簡歷、面多多、AI 導航、剪切助手、代碼小抄、算法導航、框框大學等 10 多個產品。其中 面試鴨刷題神器 (opens new window) 現在已經有幾十萬用戶，我們團隊各個產品矩陣加起來月活穩定過百萬。

    但有趣的是，我感覺自己好像又回到了原點。

    我每天還是在對著鍵盤敲敲敲：寫文章做視頻、寫編程和 AI 教程、學習新知識、調研需求寫文檔。你完全想像不到這是一個團隊管理者在做的事情，我經常有一種魂穿到兩年半之前的感覺。

    為什麼還要做這些？

    首先，我就是喜歡寫點兒東西、做點兒東西，哪怕在別人眼裡看起來是瞎折騰，我也樂在其中。

    其次，我不想讓自己只是一個管理者，而是希望保持自己的技術水平、以及持續輸出的能力。即使有一天公司真的倒閉了，我也有能力回到求職市場跟別人競爭。

    而且本來我也算是基於個人 IP 成立的公司，個人 IP 一定是我的立身之本。

    現在我們團隊的工作模式是：我負責對外輸出，團隊同學做好交付。一方面作為自己公司項目組的一員，跟團隊開發改進產品；與此同時，不斷輸出優質內容，給自家的產品帶來流量。

    我依然在持續學習 AI 等前沿方向的新技術，打造了自己的 AI 資源導航產品 (opens new window)。每天至少看 3 ~ 5 篇文章或視頻來學習新知識，這是我堅持了很多年的習慣。

    對了，去年我還開了個劇本殺店，現在已經倒閉了，哈哈哈哈哈哈哈嗚嗚嗚啊...... 這段足以寫進科幻小說、拍成短劇的經歷等後面再給大家分享吧。

    # 給想做自媒體 / 創業的朋友一些真心話

    既然聊到了自媒體和創業，分享幾點血淚經驗吧：

    1）關於冷啟動：真實最重要

    萬事開頭難。我做自媒體時，第一批粉絲完全是靠著用心的內容，獲得了一些大號的轉載機會。

    從我自己和全網的觀察來看，越真實、越自然，越容易爆火。別擔心暴露真實的自己會被噴，做自媒體被噴是很正常的，從你做 UP 主開始，就應該做好心理準備。比如我就把自己當做一個沙雕，你罵我就是你對，誰會跟沙雕較勁呢？

    總之，做好內容是一切的基礎。真實的東西在 AI 時代會更可貴。

    2）關於堅持：降低預期，持續行動

    我知道很多朋友會因為流量感到焦慮，覺得自己做的東西沒人看、沒人認可。

    但我想說：堅持下去一定會成功。

    我做自媒體這麼多年，從來沒有過真正爆火的視頻，粉絲都是從 0 開始一期一期積累來的。我剛開始在抖音做視頻，第一個視頻播放量只有 18；做公眾號時，閱讀量也是個位數。

    但我連續 100 多天下班後直播到凌晨，就是這種堅持，讓我突破了瓶頸。

    面對焦慮最好的辦法就是：降低預期並且堅持行動。把自己當做一個新人，從 0 開始用心創作內容，同時不斷改進。

    3）關於內容：有趣 > 乾貨

    摸清觀眾的心理很重要。編程深度知識或完整教程，真正要看這些內容的人會主動搜索。

    應該用輕鬆的方式，讓大家在主線學習之餘，零碎時間了解新知識。隨時記錄靈感，而不是硬著頭皮寫稿子；去除廢話、加一些梗；剪輯時加一些搞笑的動圖和音樂。

    我的內容質量把控標準是：

    • 好玩、有趣
    • 真實、共鳴
    • 要留下一些東西（知識點、一個作品）
    • 精簡、突出主旨

    4）關於產品：快速驗證，持續迭代

    不要把產品想得太
