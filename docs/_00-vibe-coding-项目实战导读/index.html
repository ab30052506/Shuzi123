              # Vibe Coding 專案實戰導讀

              從 0 到 1 構建真實專案

              你好，我是程式設計師魚皮，前騰訊全端開發，全網 200 萬粉的 AI 程式設計博主 (opens new window)，也是 AI 導航 和 程式設計導航 (opens new window) 等 10+ 自研產品的創造者。

              如果你已經看完了【基礎必讀】的內容，並且成功做出了第一個網頁應用，那麼恭喜你 —— 你已經邁出了 Vibe Coding 最關鍵的一步！

              但我猜，你現在心裡可能會有這樣的疑問：

              “做出一個簡單的待辦事項應用確實挺有成就感的，但這離真正能用的產品還差得遠吧？”

              “我想做一個更複雜的專案，但不知道從哪裡開始......”

              “看著別人做出來的那麼酷炫的 AI 應用，我也想試試，但感覺無從下手。”

              別擔心，這些想法都很正常。這個板塊就是專門為了解決這些問題而設計的。

              # 一、為什麼要做專案？

              在回答 “怎麼做專案” 之前，我想先和你聊聊 “為什麼要做專案”。

              很多同學學完基礎教程後，就開始瘋狂學各種工具、各種技術，但始終沒有做出一個完整的專案。結果就是：學了很多，但感覺什麼都不會。

              做專案，是檢驗學習成果的唯一標準。

              更重要的是，在 Vibe Coding 時代，做專案的門檻已經低到了前所未有的程度。你不需要精通所有技術細節，只需要：

              1. 知道自己想做什麼
              2. 能夠清楚地表達需求
              3. 會用 AI 工具來實現

              這三點，你在【基礎必讀】中都已經學會了。現在，是時候把這些能力應用到更有挑戰性的專案中了。

              # 二、這個板塊會講什麼？

              這個板塊的核心內容包括 專案開發的完整流程 和 各種類型的專案實戰。

              首先，我會介紹一套經過驗證的標準開發流程，從想法需求到最終實現，讓你的專案開發更加有條理。這是做好專案的基礎，強烈建議先學習這部分。

              然後，我會帶你實戰各種類型的專案。從簡單的個人工具開始，比如作品集網站、待辦事項應用、Markdown 筆記應用等，這些專案實用性強，適合新手入門。接著是熱門的 AI 應用，包括聊天助手、寫作助手、圖片生成器等，讓你掌握 AI 應用開發的核心技能。再往後是更複雜的全端應用，比如部落格系統、問答社群、線上商城等，學習如何處理前後端和資料庫。還有微信小程式的開發，從開發到上線的完整流程。

              專案做完了，還要學會如何部署上線。我會教你多種部署方式，從最新的 AI 自動部署，到最流行的 Vercel 一鍵部署，再到專業的伺服器部署，讓你的專案能夠被全世界訪問。

              最後，如果你不知道做什麼專案，我還準備了 100 個專案創意供你參考，總有一個適合你。

              此外，這個板塊還包含一些擴展內容，比如企業專案開發流程、更多企業級 AI 專案介紹，以及魚皮的原創專案實戰教程等，可以根據需要選擇性學習。

              # 三、如何使用這個板塊？

              這個板塊的文章不需要全部學完，你可以根據自己的情況靈活選擇。

              如果你是完全新手，我建議先學習《Vibe Coding 專案開發流程》，了解專案開發的基本方法。然後從個人工具開發開始實踐，選擇一個簡單的專案動手做。這樣循序漸進，不會感到太難。

              如果你對某個領域特別感興趣，也可以直接跳到對應的章節。比如你對 AI 應用感興趣，可以直接學習 AI 應用開發；你想做小程式，可以直接看小程式開發的內容。每篇文章都是相對獨立的，不看前面的內容也能理解。

              如果你不知道做什麼專案，可以先看看專案靈感大全，裡面有 100 個專案創意，總有一個能激發你的興趣。

              對於想深入學習的同學，可以在完成基礎專案後，繼續學習《企業專案開發流程》，了解真實的商業專案是如何開發的。或者學習 魚皮的原創專案 (opens new window)，跟著完整的影片和圖文教程做出完整的企業級專案。

              # 四、學完這個板塊，你能收穫什麼？

              通過這個板塊的學習和實踐，你將能夠：

              1. 掌握一套完整的專案開發流程
              2. 獨立完成從想法到產品的全過程
              3. 積累至少 2 ~ 3 個可以展示的作品
              4. 理解如何用 AI 解決實際問題
              5. 建立做專案的信心和方法論

              更重要的是，你會發現：原來做專案可以這麼簡單、這麼有趣！

              # 寫在最後

              我知道，看到 “專案實戰” 這幾個字，很多同學會覺得有點緊張。但相信我，在 Vibe Coding 的幫助下，做專案真的沒有你想像的那麼難。

              還記得我在【基礎必讀】中說過的嗎？2025 年的今天，是人類歷史上學習程式設計最容易的時刻。而做專案，也是如此。

              你不需要成為技術專家，不需要記住所有的程式碼語法，你只需要：

              • 有一個想做的東西
              • 能夠清楚地表達你的需求
              • 願意花時間去嘗試和迭代

              這三點，你都已經具備了。

              所以，準備好了嗎？讓我們一起開始這段激動人心的專案實戰之旅吧！

              加油，你一定可以的！💪
