            # AI 編程工具大全

            選擇適合你的 AI 編程工具

            你好，我是程式員魚皮，前騰訊全端開發，全網 200 萬粉的 AI 編程博主 (opens new window)，也是 AI 導航 和 編程導航 (opens new window) 等 10+ 自研產品的創造者。

            如果你已經完成了【基礎必讀】的學習，並且成功做出了第一個網頁應用，那麼恭喜你，已經體驗到了 Vibe Coding 的魅力！

            但你的心裡可能會有這樣的疑問：

            • 網頁 AI 生成器挺好用的，但我聽說 Cursor 更強大，到底該用哪個？
            • 網上說的 Claude、ChatGPT、Gemini，這些 AI 模型有什麼區別？
            • 有的工具免費，有的要付費，我應該怎麼選擇？

            別擔心，這些困惑都很正常。面對市面上幾十上百種 AI 編程工具，就算是專業程式員也會感到眼花繚亂。

            這個板塊，就是專門為了解決這些問題而設計的。

            # 一、為什麼要了解編程工具？

            在傳統編程時代，工具的選擇其實沒那麼重要。無論你用 VS Code 還是 Sublime Text，寫出來的程式碼都是一樣的。

            但在 Vibe Coding 時代，選對工具，可能讓你的開發效率提升 10 倍！

            為什麼這麼說？

            因為不同的 AI 編程工具：

            1. 能力差距很大：有的工具只能生成簡單程式碼，有的能幫你做整個專案
            2. 適用場景不同：有的適合做原型，有的適合做產品，有的適合學習
            3. 成本差異明顯：有的完全免費，有的每月要幾百塊
            4. 學習難度不同：有的上手就能用，有的需要一定基礎

            選對了工具，你可以事半功倍；選錯了工具，可能會讓你覺得 “Vibe Coding 也不過如此”。

            # 二、AI 編程工具的 3 大類型

            在深入了解具體工具之前，我們先來看看 AI 編程工具的基本分類。根據使用方式和複雜度，我把它們分為 3 大類：

            # 零程式碼平台

            在瀏覽器裡打開就能用，不需要安裝任何軟體，不需要懂任何程式碼。適合完全零基礎的新手、想快速做出原型的同學。

            代表工具：Bolt.new、Lovable、秒噠

            優勢：上手快、所見即所得、自動部署

            局限：功能相對簡單，複雜專案可能力不從心

            # AI 程式碼編輯器

            需要下載安裝，介面像傳統程式碼編輯器，但內建了強大的 AI 助手。適合有一定基礎、想深入學習 Vibe Coding、需要做複雜專案的人。

            代表工具：Cursor、Windsurf、Antigravity、Augment Code

            優勢：功能強大、靈活度高、適合大型專案

            局限：需要一定學習成本，對新手不夠友好

            # 命令行工具

            在終端裡通過命令行和 AI 對話，適合有編程基礎的開發者、喜歡命令行的極客。

            代表工具：Claude Code、Gemini CLI

            優勢：效率極高、自動化程度強、成本可控

            局限：需要一定技術基礎，新手不建議使用

            這個板塊的文章不需要全部學完，你可以根據自己的情況來選擇：

            • 如果你是完全新手：建議先看《AI 模型選擇指南》，然後重點學習《零程式碼平台》快速上手。
            • 如果你想深入學習：可以直接跳到《AI 程式碼編輯器》，學習 Cursor 的使用方法。
            • 如果你有編程基礎：可以看看第《命令行工具》，嘗試更高效的開發方式。
            • 如果你不知道選什麼：直接看《我的工具組合推薦》，我會根據不同場景給你具體建議。
            • 如果你想深入了解某個工具：可以閱讀相應的補充文章和實戰案例。
