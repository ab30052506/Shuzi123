      # Spring AI - AI 超級智能體專案實戰

      本專案是一個完整的 AI 超級智能體教程，帶你開發 AI 戀愛大師應用和擁有自主規劃能力的超級智能體。通過這個專案，你可以掌握新時代程式設計師必知必會的 AI 核心概念、AI 實用工具、AI 程式設計技術、AI 框架原理、AI 調優技巧。

      本專案採用人工編碼 + Vibe Coding 結合的方式開發，通過寫 Java 後端 + Spring AI 程式碼整合 AI、使用 Cursor 等 AI 工具輔助編碼。重點在於學習企業 AI 應用開發、AI 智能體的核心概念和實現原理、ReAct 模式、工具調用、MCP 等前沿技術。

      適合有一定專案經驗、想系統學習 AI 智能體開發、並且在履歷上添加 AI 專案來提升競爭力的同學。

      💡 專案難度較大，包含影片教程（幾十小時）、文字教程（10 萬多字）、現成的履歷寫法和面試題解。完整學完預計花費 20 小時及以上，不建議沒有後端程式設計基礎的朋友學習。

      專案程式碼免費開源：https://github.com/liyupi/yu-ai-agent

      專案文字 + 影片教程：https://www.codefather.cn/course/aiagent

      # 專案介紹

      以 AI 開發實戰 為核心的專案教程，將通過開發 AI 戀愛大師應用 + 擁有自主規劃能力的超級智能體，帶大家掌握新時代程式設計師必知必會的 AI 核心概念、AI 實用工具、AI 程式設計技術、AI 框架原理、AI 調優技巧，大幅增加求職的競爭力！

      AI 戀愛大師應用 可以依賴 AI 大模型解決用戶的情感問題，支援多輪對話、基於自定義知識庫進行問答、自主調用工具和 MCP 服務完成任務，比如調用地圖服務獲取附近地點並制定約會計劃。

      # 2、專案收穫

      本專案選題新穎、業務真實，區別於增刪改查的 “爛大街” 專案，魚皮會帶你實戰大量新技術和企業應用場景，用一套實戰教程將程式設計師必知必會的 AI 技術一網打盡，幫你成為 AI 時代企業的香餑餑，給你的履歷和求職大幅增加競爭力。

      • 各種新概念：如多模態、智能體工作流、A2A 協議、大模型評估等

      # 技術選型

      專案以 Spring AI 開發框架實戰為核心，涉及到多種主流 AI 客戶端和工具庫的運用。

      • Java 21 + Spring Boot 3 框架
      • ⭐️ Spring AI + LangChain4j
      • ⭐️ RAG 知識庫
      • ⭐️ PGvector 向量數據庫
      • ⭐ Tool Calling 工具調用
      • ⭐️ MCP 模型上下文協議
      • ⭐️ ReAct Agent 智能體構建
      • ⭐️ Serverless 計算服務
      • ⭐️ AI 大模型開發平台百煉
      • ⭐️ Cursor AI 程式碼生成
      • ⭐️ SSE 異步推送
      • 第三方接口：如 SearchAPI / Pexels API
      • Ollama 大模型部署
      • 工具庫如：Kryo 高性能序列化 + Jsoup 網頁抓取 + iText PDF 生成 + Knife4j 接口文件
